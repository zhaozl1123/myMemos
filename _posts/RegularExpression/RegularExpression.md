# 正则表达式

```bash
# [ABC]
查找[]中的所有字符
# [^ABC]
查找除去[]以外的所有字符
# [A-Z]
匹配[]所有范围内大写字符
# .
匹配除 \n \r 以外的所有字符
# [\s\S]
\s匹配所有空白符，包括\n，\S匹配所有空白符，不包括\n
# [A-Za-z0-9_]
匹配所有字母、数字、下划线
# *
ab*cde：表示匹配具有0个或n个b的字符串
# +
ab+cde：表示匹配具有1个或n个b的字符串
# ?
ab?cde：表示匹配具有0个或1个b的字符串
# {n}
ab{2}cde：表示匹配具有2个b的字符串
# {n,}
ab{2,}cde：表示匹配至少具有2个b的字符串
# {n,m}
ab{2,3}cde：表示匹配至少具有2个b最多具有3个b的字符串

```
